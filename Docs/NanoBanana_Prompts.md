# NanoBanana Note Skin Prompts

Use these prompts with the **NanoBanana** model to generate high-quality note skin assets.

## Asset Requirements
- **Target File**: `Note_Skin.png`
- **Recommended Size**: 512x256 (2:1 aspect ratio) or 256x128
- **Format**: PNG (Transparent background preferred, or Black background for Additive blending)

## Option 1: Cyberpunk Neon Bar (Recommended)
> **Prompt:**
> a sleek futuristic rhythm game note skin, horizontal bar shape, glowing neon cyan and magenta borders, high-tech glass interface texture, holographic digital glitch effect, minimalist ui element, 2d game sprite, black background, high contrast, sharp details --no gradient --ar 2:1

## Option 2: Crystal Gem (Vibrant)
> **Prompt:**
> a glowing crystal rhythm game note, rectangular gem cut, shiny reflective surface, neon blue inner glow, magical energy radiating, high quality 2d game asset, dark background, vector style, clean lines --no shadow --ar 2:1

## Option 3: Mechanical Plate (Heavy)
> **Prompt:**
> a sci-fi mechanical rhythm game note, metallic rectangular plate with glowing orange LED strips, industrial armor plating style, heavy metal texture, futuristic hud element, 2d sprite, detailed, dark background --no blur --ar 2:1

## Option 4: Retro Pixel (8-bit Style)
> **Prompt:**
> pixel art rhythm game note, neon blue and pink, retro arcade style, 16-bit graphics, glowing rectangular bar, simple and clean, black background --no anti-aliasing --ar 2:1

---

## How to Apply
1. Generate the image using one of the prompts above.
2. Remove background if necessary (keep the glow!).
3. Save as **`Note_Skin.png`**.
4. Place the file in: `Assets/Resources/AIBeat_Design/Gameplay/`.
